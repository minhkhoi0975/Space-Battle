General:
+ ESC: Exit the game
+ R: Start a new round when either of the astronauts is killed

Blue Astronaut:
+ A: Go left
+ D: Go right
+ W: Go up
+ S: Go down
+ SPACE: Shoot

Orange Astronaut:
+ Left Arrow: Go left
+ Right Arrow: Go right
+ Up Arrow: Go up
+ Down Arrow: Go down
+ ENTER: Shoot
